# Restaurant CMS

A full-featured Content Management System for managing restaurant listings, built with PHP, PDO, and Bootstrap.

## Features
- Admin Dashboard with CRUD operations
- User Authentication & Role Management
- Image Upload & Processing
- Comment System with Moderation
- Advanced Search & Filtering
- Git Version Control & Composer Management

## Technologies
- PHP 7.4+
- MySQL (PDO)
- Bootstrap 5
- TinyMCE (WYSIWYG)

## Installation
1. Clone the repository.
2. Configure `admin/db.php` with your database credentials.
3. Import the SQL schema (if available).
4. Run via XAMPP/Apache.
